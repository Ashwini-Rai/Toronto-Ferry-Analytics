# Real-Time Ferry Ticket Sales & Redemption Analytics

## Project Overview
This project analyzes ferry ticket sales and redemption patterns for Toronto Island Park using Power BI and DAX analytics.

## Objectives
- Analyze passenger movement
- Identify peak demand windows
- Improve operational planning
- Monitor real-time KPIs

## Tools Used
- Power BI
- DAX
- Excel
- CSV Dataset

## Dashboard Features
- KPI cards
- Sales & redemption trends
- Heatmaps
- Gauge charts
- Interactive slicers

## Dataset
The dataset contains:
- Timestamp
- Sales Count
- Redemption Count
- Unique IDs
